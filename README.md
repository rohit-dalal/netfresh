# netfresh
# A simple bash command tool to fix network related problems

`Usage: /usr/bin/netfresh.sh [flags]`

`Flags:`

`-r, --restart        Restart all network services.`
        
`-s, --start          Start all network services.`
        
`--stop               Stop all network services.`
